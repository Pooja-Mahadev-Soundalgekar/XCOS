Name : Change Properties
Description : Java Program to allow user to change the properties of a label using Swing.

Instructions : 
Compile all the files and run MainPage.java.

Right-clicking the 'Change Properties' label opens up a popup which contains three options.

Click the 'Properties' item
A new window opens up with fields to change the text, background color and foreground color. Maximize the window to see all the fields.

Enter the values for colors in Hex (#ff0000 for red) and click submit.              

File list and description :                          "MainPage.java" - File containing main method that executes the JFrame class.

"PopupBox.java" - The first window which contains the label 'Change Properties'.

"PopUpWindow.java" - Popup window which contains text fields for inputting the 3 values mentioned.
