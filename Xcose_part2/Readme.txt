Name : Student Ranking
Description : Java Program to input details of students and then generate a rank list based on the marks obtained.

Instructions : 
Compile all the files and run RankStudents.java.

Follow on-screen instructions to input student details.        

File list and description :
"RankStudents.java" - File containing main method that inputs details and outputs the ranklist.

"Student.java" - Class containing all the properties of a student along with getter and setter methods.
